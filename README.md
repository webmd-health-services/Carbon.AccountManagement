# System Requirements

* Windows PowerShell 5.1 and .NET 4.6.1+
* PowerShell Core 6+

# Installing

To install globally:

```powershell
Install-Module -Name 'Carbon.AccountManagement'
Import-Module -Name 'Carbon.AccountManagement'
```

To install privately:

```powershell
Save-Module -Name 'Carbon.AccountManagement' -Path '.'
Import-Module -Name '.\Carbon.AccountManagement'
```
